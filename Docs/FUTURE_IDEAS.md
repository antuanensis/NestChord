# Future Ideas

- Style-aware voicing packs for pads, stabs, plucks, bass-led progressions, and cinematic harmony.
- Borrowed chord lanes with visual indicators for source mode.
- Harmonic rhythm tools such as Euclidean spacing and polymetric loops.
- Bass motion constraints independent of upper voices.
- Pattern morphing between two harmonic timelines.
- Per-event humanization, strum direction, strum width, and note-order presets.
- JSON import/export for generated and shared patterns.
- Optional natural language composer that outputs validated `Pattern` files.
- Host automation for energy, density, inversion, register, and probability.
- Gesture editing: reorder by dragging blocks, resize with an edge handle, and audition by double tapping.
- Visual harmonic density lanes for tension, register, and rhythmic activity above the Chord Blocks.
