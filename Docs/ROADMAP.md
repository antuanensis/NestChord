# Roadmap

## Phase 1: Foundation MVP

- AUv3 MIDI plugin shell.
- Host sync.
- Basic pattern model.
- Degree-based chord sequencing.
- Basic SwiftUI editor.
- MIDI output bridge.
- State save/restore.
- Unit tests.

## Phase 2: Better Composition Workflow

- Drag-and-resize harmonic blocks.
- Better grid UI.
- Presets.
- Pattern duplication and randomization.
- More quantization grids.
- Per-event probability.
- Event mute/solo.
- Multiple patterns.

## Phase 3: Advanced Harmony

- Borrowed chords.
- Secondary dominants.
- Modal interchange.
- Negative harmony.
- Chord substitutions.
- Slash chords.
- Pedal tones.
- Custom scales.
- User-defined chord dictionaries.

## Phase 4: Voice Leading Intelligence

- Smooth voice leading.
- Common-tone retention.
- Minimal movement algorithm.
- Register constraints.
- Bass movement rules.
- Style presets: classical, jazz, ambient, techno stabs, trance pads, cinematic, dub chords.

## Phase 5: Generative Features

- Probability per event.
- Markov chord transitions.
- Tension/release curves.
- Energy automation.
- Euclidean harmonic rhythm.
- Polymetric chord loops.
- Generative variation per loop cycle.
- Humanization, strumming, arpeggiation, and pattern morphing.

## Phase 6: AI / Natural Language Layer

- Prompt-to-pattern generation.
- JSON pattern import/export.
- Validation of generated harmonic structures.
- Style templates.

Natural language should generate harmonic event timelines, not raw MIDI notes.
